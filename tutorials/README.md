# Integration tutorial videos

Short, step-by-step motion-graphic walkthroughs that show public users how to
connect each integration to Neuron. They match the visual language of the
marketing clips (dark navy stage, eyebrow + bold headline, a floating product
window, an animated cursor that clicks through the real flow, footer caption).

Each walkthrough is grounded in the real product flow — actual button labels,
OAuth scopes, and screens from `app/(dashboard)/dashboard/integrations`.

## Videos (`out/`)

| File | Flow |
| --- | --- |
| `connect-slack.mp4` | Integrations → Connect → Slack OAuth (Allow) → Sync Now |
| `connect-linear.mp4` | Integrations → Connect → Linear OAuth (Authorize) → Sync Now |
| `connect-gmail.mp4` | Set up → read-only Google consent → Labels → Filters → Review → Start Sync |
| `connect-notion.mp4` | Set up Notion → Choose pages → **add the Neuron connection in Notion** → Sync Now |

> Note: `connect-notion.mp4` is generated by the same pipeline but was not
> included in the initial commit. Run `./build.sh notion` to produce it.

All clips render at **1120×760, 30fps, H.264** to match the existing marketing videos.

## How it works

The renderer is a small, data-driven engine — rendering is a pure function of
time (`window.seek(t)`), so frames are captured deterministically.

- `style.css` — shared visual system (stage, window chrome, cards, OAuth screens, cursor).
- `engine.js` — builds the DOM from `window.NEURON_VIDEO` and implements `seek(t)`:
  scene crossfades, staggered reveals, and the animated cursor/click.
- `frame.html` — host page; loads `?v=<name>.js` then `engine.js`.
- `slack.js` / `linear.js` / `gmail.js` / `notion.js` — per-video scene scripts (data only).
- `capture.mjs` — drives headless Chromium, calls `seek(t)` per frame, screenshots.
- `build.sh` — captures frames then encodes to MP4 with ffmpeg.

## Regenerate

Requires Node, a Chromium (Playwright's cached build works), and ffmpeg.

```bash
cd tutorials
npm install            # playwright-core
export PW_CHROMIUM="/path/to/chromium"   # see build.sh for the default
./build.sh slack       # -> out/connect-slack.mp4
./build.sh linear
./build.sh gmail
./build.sh notion
```

To change copy, timing, or steps, edit the scene script for that integration —
no engine changes needed.
